# Lal's Motor Winders (Fiji) Pte Limited

Public website for **Lal's Motor Winders** — motor rewinding, generator servicing, and electrical contracting based in Labasa, Fiji.

## Design system

Generated with [UI UX Pro Max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill):

- Pattern: Trust & Authority + Conversion
- Colors: industrial slate + stock green
- Typography: Poppins / Open Sans

See `design-system/lal-s-motor-winders/MASTER.md`.
