# Lal's Motor Winders Fiji PTE Limited

Professional website for Lal's Motor Winders Fiji — expert electrical motor rewinding, pump repair, and industrial maintenance services across Fiji.

## Features

- Responsive landing page built with HTML + Tailwind CSS
- [UI UX Pro Max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) design system integration
- Trust-focused layout with services, about, and contact sections
- Accessible UI with keyboard navigation and reduced-motion support

## Getting Started

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).

## Design System

Design tokens and guidelines are persisted in `design-system/lal-s-motor-winders-fiji/MASTER.md`, generated by the UI UX Pro Max skill.

## UI UX Pro Max Skill

This project includes the [UI UX Pro Max](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) Cursor skill in `.cursor/skills/`. It activates automatically when you request UI/UX work.

To update the skill:

```bash
npx ui-ux-pro-max-cli update
```

## Customization

Update contact details in `index.html` (phone, email, address) with your actual business information.
