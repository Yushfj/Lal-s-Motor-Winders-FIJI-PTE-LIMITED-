# Design System — Lal's Motor Winders Fiji

> Motor windings floating on black velvet

**Theme:** dark  
**Base system:** [Dala](https://dala.ai) constellation aesthetic

## Implementation

Full Dala token set applied in `css/styles.css`:
- Pure `#000000` void canvas — no gray panels, cards, borders, or shadows
- Electric Iris `#8052ff` — sole filled button color
- Saffron Spark `#ffb829` — amber labels and accent links
- **Syne** display + **Outfit** body (expressive, non-default stacks)
- Display scale with aggressive negative tracking; brand-hero “Lal's” leads the first viewport
- Spacing base unit: 6px (6–120px scale)
- Button radius: 22.5px pill

## Signature Visual

Dense triangular particle constellation forming electrical shapes (stator → generator → lightning → transformer → spark). Mobile uses a lighter particle budget and no bloom.
